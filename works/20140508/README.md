# 本日の課題

## Procの使い方

* 現在時間を返すProcを作り、呼び出して下さい

* 引数を3倍にして返すProcを作り、呼び出して下さい

* 引数で配列を受け取り、各値を3倍にした配列を返すProcを作り、呼び出して下さい

* クラスに対して上記Procを渡し、`hoge`メソッドの引数に対してProcを呼び出して返すようにして下さい
  ~~~
  x = TestClass.new
  x.custom_proc = Proc.new{...}
  x.hoge [1,2,3]
  # =>  [3,6,9]
  ~~~

* 前述のProcを、各値を5倍するものに変えて下さい
  ~~~
  x.custom_proc = Proc.new{...}
  x.hoge [1,2,3]
  # =>  [5,10,15]
  ~~~
